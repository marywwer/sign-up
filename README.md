# 🌐 Личный сайт / Frontend-проект

Простой одностраничный сайт, представляющий регистрацию с пометкой "надежности" пароля. С визуальной точки зрения сайт не закончен.

## 📁 Структура проекта
- `sign-up.html` — Главная страница
- `/css/` — Стилизация
- `/js — Скрипты
- `/img/` — Изображения

## 🛠 Использовано:
- HTML5
- CSS3
- JavaScript (ES6)
